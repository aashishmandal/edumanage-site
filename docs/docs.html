Documentation

Step-by-step guide to install, configure, and use EduManage.

Installation

  • Download the latest version from the Download page.
  • Extract the ZIP file (if applicable).
  • Double-click the EduManage executable to launch the app.

Configuration

  • Set your school name and logo in the Settings tab.
  • Assign roles to users: Viewer, Editor, Admin.
  • Enable Test Mode for safe testing without saving data.

Modules

  • Students: Add, update, delete, and export student records.
  • Documents: Upload and manage student files securely.
  • Reports: Generate PDF/CSV exports for selected students.
  • History: View attendance and payment logs.

Security

  • Role-based access ensures only authorized users can modify data.
  • Test Mode prevents accidental changes during training or demos.
  • All data is stored locally in a secure SQLite database.

Backup & Restore

  • Use the Backup tab to export your database safely.
  • Restore from a previous backup if needed.
